# Front-End-Project-Sagra-Locale-Web
Progetto web realizzato come esercizio di front-end: sito informativo e promozionale dedicato alla Sagra U’ Sfriuonzolo Sotto l’Arco. Include homepage con carosello, news, menù dei piatti tipici, galleria immagini e mappa interattiva. Realizzato con HTML, CSS e JavaScript vanilla, modulare e responsive.
